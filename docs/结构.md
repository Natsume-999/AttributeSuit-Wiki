## 文件结构

目录 `AttributeSuit` 被删除时，会重新加载配置文件。  

|  文件 / 文件夹   | 说明  |
| :--------- | :--------- |
| project/            | 套装处理目录            |
| config.yml          | 主配置文件              |
| lang.yml            | 语言文件                |

## 变量结构

注意，由于是同步获取变量内容，不要重复执行过多次!  

|  变量   | 说明  |
| :--------- | :--------- |
| %attributeSuit_suit_<套装>%   | 获取某个套装激活状态 (返回 true | false 两种类型) |
| %attributeSuit_number_<套装>%   | 获取某个套装激活数量 |
| %attributeSuit_all_suit%   | 获取已激活的全部套装数量 |
